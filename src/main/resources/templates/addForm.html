Для того, чтобы добавить рецепт, впишите в соответсвующие поля название блюда, список ингредиентов для его приготовления, подготовительные инструкции к готовке, если они требуются, и сам рецепт блюда. Важное замечание: для корректной обработки рецепта необходимо писать пункты без перехода на новую строку, то есть нажатия символа "enter"
